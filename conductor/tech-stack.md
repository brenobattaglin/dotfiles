# Technology Stack: dotfiles

## Core Infrastructure
- **Shell (Bash/Zsh):** The primary scripting language used for all installation and setup tasks.
- **Makefile:** Provides a unified interface for running setup commands.

## Package Management
- **Homebrew:** Used for system-wide application and utility management on macOS.
- **npm/nvm:** Managing Node.js versions and packages.
- **Python/Pip:** Managing Python environments and dependencies.
- **dotnet:** Support for .NET development environment setup.

## Configuration and Automation
- **Zsh Aliases & Oh-My-Zsh:** Personalized shell configurations for increased productivity.
- **Git Config:** Standardized Git configuration for all projects.
- **macOS Defaults (via AppleScript/Shell):** Automating system preference adjustments for a consistent experience.
